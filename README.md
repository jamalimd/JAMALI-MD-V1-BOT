- AI Chat

- Gemini AI
- Logo Maker
- Downloaders
- Group Manager
- Auto Status View
- Auto React
- Auto Follow Channel
- Auto Join Groups
- Anti Delete
- Anti Link
- Anti Call
- Auto Typing
- Auto Recording
- Welcome & Goodbye
- Sticker Maker
- QR Scanner
- URL Shortener
- Weather
- Calculator
- Dictionary
- Translation
- Games
- Memes
- Quotes
- Owner Commands
- Admin Commands
- Utility Commands
- 1296+ Commands
